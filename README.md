# Shannon Game
An experience to show human cannot produce randomness.

## To setup
`mkdir new_directory`
`cd new_directory`
`git clone https://github.com/7bmax/shannon_game.git`
`python3 -m venv .venv`
`source .venv/bin/activate`
`pip install -r requirements.txt`
`cd game`
`streamlit run webapp.py`
